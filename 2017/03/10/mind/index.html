CSS

flex父容器属性 ( 加粗为默认属性 )

  • flex-direction 方向( row | row-reverse | column | column-reverse ) 主轴 | 主轴倒序 | 侧轴 | 侧轴倒序
  • flex-wrap 是否换行 ( nowrap | wrap | wrap-reverse ) 不换行 | 换行 | 换行方向和轴方向相反
  • flex-flow 以上两条属性的合集
  • justify-content 主轴方向上的对齐方式 ( flex-start | flex-end | center | space-between | space-around ) 起点 | 终点 | 居中 | 两端 | 等外边框
  • align-items 侧轴方向上的对齐方式 ( flex-start | flex-end | center | baseline | stretch ) 起点 | 终点 | 居中 | 文字基线 | 延伸到两端
  • align-content 多跟主轴对齐方式( 不常用 ) flex-start | flex-end | center | space-between | space-around | stretch

flex子容器属性 ( 加粗为默认属性或值 )

  • order 排序数字 ( int ) 0
  • flex-grow 成长空间、比例 ( number ) 0
  • flex-shrink 缩减空间、比例 ( number ) 1
  • flex-basis 占据主轴空间 ( length | auto ) auto
  • flex 以上三条属性的合集
  • align-self 自己的侧轴对齐方式 auto | flex-start | flex-end | center | baseline | stretch
